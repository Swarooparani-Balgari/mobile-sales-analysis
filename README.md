Mobile Sales Analysis

Tools Used
- Excel
- SQL
- Python (Pandas)

Dataset
Mobile phone sales dataset containing brand, city, salesperson and revenue information.

Analysis Performed
- Revenue calculation
- Revenue by brand

- Sales dashboard

Key Insights
- Apple generated highest revenue
- Delhi produced the most sales
- Rahul was the top salesperson
- On the day 01 June 2025 had the highest revenue among all days
